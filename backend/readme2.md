# Implementation Guide

### 1. Prerequisites
- Node.js installed
- PostgreSQL installed and running

### 2. Setup Steps
```bash
# Navigate to backend folder
cd backend

# Install dependencies
npm install

# Build TypeScript code
npm run build

# Start Development Mode
npm run dev
```

### 3. Configuration (.env)
Update the `.env` file with your local PostgreSQL credentials:
- `DB_USER`: Your postgres username
- `DB_PASS`: Your postgres password
- `DB_NAME`: Create a database named `ravankargah_db` in your Postgres

### 4. Database Migration
The server is configured to automatically sync and create tables on the first run using `sequelize.sync({ alter: true })`. Just start the server and the tables will be generated.

### 5. API Endpoints
- `POST /api/auth/register`: Expects full user profile.
- `POST /api/auth/login`: Needs `mobile` and `nationalCode`.
- `POST /api/mcmi/submit`: Saves JSON report generated by frontend.
- `GET /api/mcmi/my-results`: Retrieves testing history for the logged-in user.
